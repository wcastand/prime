# Prime table

# Try it

Install the dependencies :

```
$ npm install
// or
$ yarn
```

Launch the server :

```
$ npm run start
// or
$ yarn start
```

Launch the test :

```
$ npm test
// or
$ yarn test
```

To get watch tests : `$ npm testdev`

You can regenerate a prime table by clicking the small pencil in the top left of the page.
